# Data-Science-Coding

A collection of data science programming solutions implemented in R. This repository contains coding exercises and problem-solving approaches focused on data analysis, statistical computing, and data science concepts.

# Overview

This repository is organized into separate question folders, each containing solutions and supporting code for a specific data science challenge.

# Question 1 Folder (DS)

Solution and analysis for generating SDTM DS by using the package sdtm.oak. The folder contains the program, output, and log file.

# Question 2 Folder (ADSL)

Solution and analysis for generating ADSL. The folder contains the program, output, and log file.

# Question 3 Folder (AE Table And Graphs)

Solution and analysis for generating Treatment-Emergent Adverse Events and 2 AE Graphs. The folder contains the program, output, and log file.
